# 📝 Text Editor with Undo/Redo

A command-line text editor built in C++ using two stacks to manage undo and redo functionalities, mimicking real-world editor behavior.

## ✨ Features
- ✍️ Insert and delete text
- ↩️ Undo previous actions
- ↪️ Redo undone actions
- 📄 Live preview of current text
- 📚 Command-based CLI system

## 💡 Technologies Used
- C++
- Stack (LIFO)
- OOP principles
